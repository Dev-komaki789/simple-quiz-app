# simple-quiz-app

6 カテゴリ × 10 問の 4 択クイズアプリ。各問 10 秒の制限時間付き。

## 機能

- カテゴリ選択（日本史 / 理科 / 地理 / 雑学 / 漫画 / アニメ）
- 各カテゴリ 10 問、ランダム出題
- 1 問につき 10 秒のカウントダウンタイマー
- 正誤判定（〇 / ×）の即時表示
- 結果画面（スコア + 全問のおさらい）

## 技術スタック

HTML / CSS / Vanilla JavaScript（ビルドツール無し、`index.html` を直接開けば動く）

## 構成

- 問題データは JSON ファイル（カテゴリごとに 1 ファイル）で管理、fetch で非同期に読み込み
- 状態管理は素の変数（フレームワーク不使用）
- 画面遷移は DOM の `display` 切替で実装

## メモ

WMS や EC を作る前の学習段階で、JSON データのハンドリング・非同期処理 (fetch)・
タイマー処理・複数画面の遷移を素の JS で組むために作成した小品です。
